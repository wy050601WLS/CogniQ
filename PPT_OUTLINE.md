## 幻灯片 3：项目概述

**标题**：01 项目概述

**内容**：
知识问答系统是一个基于 RAG（检索增强生成）技术的智能知识问答平台。用户可以上传文档到知识库，系统会自动解析、分块、向量化并存储，然后通过 AI 模型对用户的问题进行检索增强生成回答。

**核心价值（4个卡片）**：
- 📚 知识管理：支持 PDF/Word/MD/TXT/HTML 多种格式文档上传与解析
- 🤖 智能问答：基于 RAG 技术，AI 根据知识库精准回答
- 📖 来源引用：每个回答标注参考来源，确保信息可追溯
- 💬 多轮对话：支持上下文记忆，理解用户追问意图

## 幻灯片 6：RAG 核心流程

**标题**：03 RAG 智能问答流程

**流程步骤**：
1. 用户提问 → 输入问题文本
2. 向量化 → Embedding 模型转换为 512 维向量
3. 向量检索 → ChromaDB 余弦相似度 Top-5
4. 构建 Prompt → 注入检索结果 + 对话历史
5. LLM 生成 → Ollama Qwen2 流式输出
6. 返回回答 → 附带来源引用，实时渲染

**RAG 优势**：
- 减少幻觉：基于真实文档回答，降低 AI 编造信息
- 知识可更新：上传新文档即可，无需重新训练模型
- 来源可追溯：每个回答标注来源，用户可验证
- 成本低：只需存储文档向量，无需 GPU 训练

## 幻灯片 9：安全设计

**标题**：05 安全设计

**认证安全**：
- JWT 双 Token 认证（Access 60min + Refresh 7d）
- Token 轮换机制，Refresh Token 使用后失效
- PBKDF2 密码哈希 + 随机盐值（100,000 次迭代）
- hmac.compare_digest 防止时序攻击
- 基于 IP 的登录/注册频率限制

**数据安全**：
- XSS 防护：DOMPurify 消毒所有 HTML 输出
- SQL 注入防护：SQLAlchemy ORM 参数化查询
- 文件上传验证：类型、大小限制
- CORS 配置：限制跨域来源

**运行时安全**：
- 生产环境禁用 Swagger UI
- JWT 密钥强制配置检查
- 错误信息脱敏，不暴露内部细节
- 操作日志审计

## 幻灯片 10：技术难点与解决方案

**标题**：06 技术难点与解决方案

**难点 1：向量检索与事件循环冲突**
- 问题：ChromaDB 同步查询阻塞 FastAPI 事件循环
- 方案：asyncio.to_thread() 线程池执行

**难点 2：流式 Session 作用域**
- 问题：StreamingResponse 消费时 session 已关闭
- 方案：生成器内部创建独立 session

**难点 3：SSE 分片解析**
- 问题：数据跨 chunk 到达，需正确拼接
- 方案：buffer 机制保留不完整行

**难点 4：多轮对话上下文**
- 问题：LLM 上下文窗口有限
- 方案：最近 5 轮历史 + 消息截断
